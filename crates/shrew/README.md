# shrew

The declarative tensor language and engine.

This crate serves as the main entry point to the library, re-exporting modules from:
- `shrew-core`: Runtime and tensors
- `shrew-ir`: Compiler frontend
- `shrew-nn`: Neural network primitives

## Usage

Add this to your `Cargo.toml`:
```toml
[dependencies]
shrew = "0.1"
```
